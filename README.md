# Yukis_Army_knife

## 概要

パソコンで使える十徳ナイフを目指した、多様なツールを内包したランチャーソフトです。

ソフトに関する詳しい解説はこちらから

https://github.com/yukisakura001/Yukis_Army_knife/wiki

## 技術

### 使用言語

Pythonのみ

### 主要なライブラリ

- Tkinter（GUI作成）
- Pyinstaller（ビルド）
- Pystray（タスクトレイ常駐）
- Pillow・openCV（画像編集）
- Moviepy（動画編集）
- Threading（並列処理）
- Requests（API呼び出し）

etc ...
